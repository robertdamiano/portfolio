## Overview

A static marketing website for Bell Driver Education Inc., a ministry-approved driving school serving the Greater Toronto Area since 1954. The site features a testimonial carousel, service area highlights, course breakdowns, and a fully responsive layout.

## Technical highlights

- Static HTML/CSS/JS with no framework overhead
- Firebase Hosting for fast global delivery
- GitHub Actions CI/CD pipeline for automatic deploys on push to main
- Responsive design covering mobile through desktop

## Hard parts

The main challenge was structuring a content-heavy site to feel modern and scannable while preserving all the detail a prospective student needs — course formats, service areas, pricing, and testimonials — without reaching for a framework or CMS.
